# 🔐 Tela de Login Moderna

Projeto de tela de login desenvolvido com HTML, CSS e JavaScript.

## 🚀 Funcionalidades
- Validação de login
- Mostrar/ocultar senha
- Feedback visual de erro e sucesso
- Animação de erro (shake)
- Interface moderna com efeito glassmorphism

## 🛠 Tecnologias
- HTML5
- CSS3
- JavaScript
